# intikasirlaundry
Aplikasi Kasir/PoS untuk Bisnis Laundry


## Download

Unduh versi terbaru di [Releases](https://github.com/stargan-id/intikasirlaundry/releases/latest).

### Instalasi
1. Download file `app-release.apk` dari halaman Releases
2. Buka file APK di perangkat Android
3. Izinkan instalasi dari sumber tidak dikenal jika diminta
4. Buat akun pemilik dan mulai gunakan

### Minimum Requirements
- Android 7.0 (API 24) ke atas
